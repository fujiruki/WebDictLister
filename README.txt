This software make a word list which linked to Weblio page. And the page is opened on the right side.
(この文はおかしいですか？教えてください...)


/* 使い方
*****************************/
1. Excelでわからない単語リスト(1列)を作成する。
2. 単語リストをコピーする。
3. ページの左上のテキストボックスへペースト。
4. 「OKボタン」
5. 意味を調べたい単語をクリックすると右にページが表示される。

「単語を１つずつ追加したい！」という方
1. 左下の小さいテキストボックスへ単語を入力。
2. Enterキーを押す。
3. 1,2を繰り返すことで連続登録。
