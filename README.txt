/*****************************
 * WebDictLister
 *
 * Weblioページへリンクのついた単語リストを作成します。
 * リンクをクリックすると同一画面でWeblioページが開きます。
*****************************/

This make a word list which have link to Weblio page. And the page is opened on the right.


/* 使い方
*****************************/
1. Excelでわからない単語リスト(1列)を作成する。
2. 単語リストをコピーする。
3. ページの左上のテキストボックスへペースト。
4. 「OKボタン」
5. 意味を調べたい単語をクリックすると右にページが表示される。
